irh

Usage

mystateEngine.irh(eventName, val, target)

Parameters

eventName Name (string) of request

val (optional) Value associated with the request

target (optional) The instance generating the event

Returns

Returns the value it receives from asking the engine's oracle, or undefined if no oracle has been set.

Description

This is the default internal request handler for the state engine. Developers must override this method if they expect to receive requests from objects controlled by this state engine. The default method merely announces that it has been called, and tries to use the engine's oracle to answer the request.
