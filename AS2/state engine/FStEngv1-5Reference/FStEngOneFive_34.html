addStMgr

Usage

myhStateC.addStMgr(sm)

Parameters

sm State (network) manager pointer

Returns

Nothing

Description

This method is used to add a state network manager to the specified hStateC. The hStateC can manage multiple concurrent networks. The only catch is that sub-states in the sub-network(s) must have unique ID's with respect to sibling states in sub-network(s).
