chgSt

Usage

mystateMgr.chgSt(transID, netLvl, val)

Parameters

transID ID of the transition to fire

netLvl Whether or not this is a transition off of the state network level (container state) into the state network

val An optional argument that is passed to the transition function.

Returns

Nothing

Description

Change the current state along transition with id # 'trans'. In version 1.5, there is little reason to call this because transitions with triggers are handled automatically.
