state

Description

This class defines a simple state. You attach the state to the state engine by specifying the state network manager pointer (of type stateMgr). If this is the topmost (root) state in the statechart, you pass in undefined (and then use the stateEngine's setStateNetwork() to attach it to the engine).

Usage

new state(name, id, stateMgrPtr);

Parameters

name String name of the state. Used in trace output

id Typically numeric, this is the ID of the state (must be unique among other states in the same network level)

stateMgrPtr Pointer to the state manager for the local network in which the state resides (not to be confused with the sub-network this state manages), or undefined for the root state

Example

// state id is 0
myState0 = new state("State Name", 0, myMgrPtr);

Code Hint

Variable names with the suffix "_state" are recognized for the sake of codehinting, as states.
