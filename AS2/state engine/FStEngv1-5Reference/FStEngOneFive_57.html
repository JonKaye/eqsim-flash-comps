addTrans

Usage

mystate.addTrans(id, trans, tf)

Parameters

id ID of the transition (must be unique for this state)

trans A trans or transLvl object instance

tf Trigger name (string) for this transition (FStEng v1.5)

Returns

Nothing

Description

Add a transition (either simple or multi-level) to this state. If the developer wants to manually code the event handler, then the developer uses chgSt() to fire this transition when it is triggered. To use the v1.5 event handling mechanism, simply tell this method the event name string (third parameter).
