addContinActivity

Usage

myhStateC.addContinActivity(id, mclip, hideClip)

Parameters

id ID for this continuous activity (must be unique only for other continuous activities on this state)

mclip Movie clip to activate/deactivate

hideClip Whether or not to make movie clip invisible when state is inactive

Returns

Nothing

Description

Adds a continuous state activity for the state. A continuous activity is simply a mechanism to start and stop a movie clip upon entry/exit from a state. Upon state entry, the movie clip is told "gotoAndPlay(2)". On exit, the clip is told "gotoAndStop(1)". There is a parameter, hideClip, that says whether or not to make the movie clip invisible (default is true) when the state is inactive.
