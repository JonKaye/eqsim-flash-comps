This class extends FMXISButtonRepeater (momentary button with repeat-on-down functionality) to allow developers to swap out the button graphics dynamically, based on specified linkage ID's of the graphics.

This class extends FMXISButtonMomentary (momentary button with repeat-on-down functionality) to allow developers to add a text label placed over the button.

This class implements a simple momentary push button that is capable of generating repeated, button down events (onPress) given a waiting period and frequency.
