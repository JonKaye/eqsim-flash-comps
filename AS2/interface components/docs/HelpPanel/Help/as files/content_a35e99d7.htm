FMXISTimer Class

Inheritance > FMXISBase > FMXISTimer

ActionScript Class Name mx.fmxis.FMXISTimer

Event generated when interval elapses. Value supplied with the event is the number of remaining iterations until shutdown (-1 is infinite).
