FMXISJoystick.moveHS() Method

Usage

fMXISJoystick.moveHS(noMouse, quiet)

Parameters

noMouse Set this to true if you are moving the joystick programmatically.

quiet Set this to true to not generate an onJStart event

Returns

Nothing.

Description

Method; Initialize moving of hot spot. Developers use this only when moving the joystick programmatically. Use setVec() to set the position, after calling moveHS(), then call stopMoveHS when you are finished moving the joystick (make sure the pass the argument as true).
