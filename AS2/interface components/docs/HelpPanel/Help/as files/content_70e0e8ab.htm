FMXISJoystick.maxVal Property

Usage

fMXISJoystick.maxVal

Description

Property, read/write; Maximum value for the vector coordinates when the joystick is at full extension. The joystick center is always 0 (for both x and y coordinates), and this property sets the maximum value for the coordinate. The maximum is used to set positive and negative extent, so joystick range will be (-maxVal,-maxVal) to (maxVal, maxVal).
