FMXISButton.evtRollOver Property

Usage

fMXISButton.evtRollOver

Description

Property, read/write; Event name of onRollOver event. The event names should really only be set at the time the component is instantiated (either at run-time, or programmatically at the attachMovie/createClassObject time). However, you can change it at any time programmatically and the component will use that event name.
