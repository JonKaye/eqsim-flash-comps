FMXISJog.incrVal() Method

Usage

fMXISJog.incrVal(incr, quietly)

Parameters

incr Numeric increment to change the jog.

quietly (optional) Set this to true to make the change without generating an event.

Returns

Nothing.

Description

Method; Increments the jog by incr units (positive or negative). This can be used to simulate turning the jog.

Example

myJog.incrVal(2.5);
