FMXISPotentiometer.setVal() Method

Usage

fMXISPotentiometer.setVal(v, q)

Parameters

v Numeric value (between 0 and units)

q (optional) Boolean flag indicating whether to set the value quietly (no event notification) or with notification (default). You can also use the val property to set the value, but it does not generate an event.

Returns

Nothing.

Description

Method; Sets the potentiometer value.

Example

myPot.setVal(5); // sets the knob to 5
