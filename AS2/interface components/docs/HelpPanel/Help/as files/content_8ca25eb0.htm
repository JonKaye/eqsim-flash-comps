FMXISPotentiometer.execEvent() Method

Usage

fMXISPotentiometer.execEvent(evName, evVal, quiet)

Parameters

evName Event name (string) must match the event this component generates

evVal (optional) value accompanying the event (if the event has an accompanying value)

quiet (optional) set this to true if component should perform the action but not generate an event (this should be false or undefined, unless you know what you are doing)

Returns

Nothing.

Description

Method; Given an event string and value that matches what this component would generate, perform the action. For the potentiometer component, the event is onValChg and the value is the new value.

Example

myPot.execEvent("onValChg", 1); // set the potentiometer to 1
